# Elder Care Vision

AI-Powered Fall Detection for Seniors.

## Installation

**Create a virtual environment:**

```bash
python -m venv .venv
# Windows PowerShell
.\.venv\Scripts\Activate.ps1
# Unix/Linux
source .venv/bin/activate
```

**Windows**:

```
install.bat
```

**Other systems**:

```
python -m pip install -U pip wheel setuptools
pip install -e .[dev]
```
